# Assets

Binary assets referenced by documentation: images, exported diagrams, screenshots.

Rules:

- Prefer Mermaid-in-Markdown over binary diagrams ([markdown conventions](../docs/standards/markdown-conventions.md));
  this directory is for what genuinely cannot be text.
- Every asset is referenced by at least one document; orphans get deleted.
- Naming: `kebab-case`, prefixed by topic (`network-topology.png`).
- No photos or personal media — that is application data and lives outside the repository.
