# Templates

Scaffolding for new artifacts. Copy, don't reinvent — templates are how the
[standards](../docs/standards/README.md) become the path of least resistance.

| Template                 | Creates                        | Governing standard                                        |
| ------------------------ | ------------------------------ | --------------------------------------------------------- |
| [`service/`](service/)   | A new service directory        | [service-structure](../docs/standards/service-structure.md) |

Document templates live next to their document type:
[ADR template](../docs/decisions/TEMPLATE.md), [runbook template](../docs/runbooks/TEMPLATE.md).

When a standard changes, its template changes in the same commit.
