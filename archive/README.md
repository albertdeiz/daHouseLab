# Archive

Retired services and superseded documents. History is preserved here instead of deleted, because
"why did we stop doing X?" is a question this repository must be able to answer.

Rules:

- Archived items are renamed `YYYY-MM-DD-original-name` (date of archival).
- Retired services arrive as their whole directory, plus a short note in this README's log:
  what it was, why retired, where its data went.
- Nothing here is maintained or expected to work; it is read-only context.
